# itsm

should support those features/functions.

1. cmdb - configuration management database
2. helpdesk
3. slm - services level management
4. change
5. knowledge base
6. billing

## 编码规范

- 遵照 Odoo 编码规则
- 以英文作为编程语言，按 Odoo 国际化翻译为中文以及其他语言

## 运行环境

以 Odoo13 为运行环境，不用支持之前版本
